Proyecto IMT2200

Introducción a la Ciencia de Datos: Predicción de precios inmobiliarios en la RM

Descripción del proyecto

En este proyecto analizamos el conjunto de datos del portal inmobiliario con el objetivo de realizar Web Scraping de los datos del portal. Utilizamos herramientas de ciencia de datos para limpiar, transformar y explorar la información para entender el mercado inmobiliario.

Datos y Metodología

  • Fuente de datos: Portal Inmobiliario.
  • ETL: Limpieza de datos, transformaciones principales y manejo de valores faltantes (Nulls/NaN).
  • Análisis: Definición de métricas, visualizaciones con Matplotlib/Seaborn y modelado de datos.
